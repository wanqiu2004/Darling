Flexbox 布局详解

Flexbox 是一种一维的布局模型，用于通过盒模型来控制子元素的对齐、排列、大小等。在 Flexbox 中，父元素被称为弹性容器，子元素被称为弹性元素。

一、Flexbox 布局基本概念

Flexbox 可以帮助我们设计响应式布局，适应不同屏幕尺寸，且不依赖固定宽高。

Flexbox 的目的是实现一维的内容分布，可以沿任意一个方向（横向或纵向）排列子元素。

二、弹性容器的属性

1. flex-direction

控制弹性元素排列的方向。

  • row (默认)：从左到右排列。
  • row-reverse：从右到左排列。
  • column：从上到下排列。
  • column-reverse：从下到上排列。

2. flex-wrap

决定弹性元素是否换行。

  • nowrap (默认)：不换行，所有弹性元素放在一行内。
  • wrap：允许弹性元素换行。
  • wrap-reverse：允许换行，且新行会在上方。

3. justify-content

控制弹性容器在主轴（水平）上的对齐方式。

  • flex-start：对齐到起始位置。
  • flex-end：对齐到结束位置。
  • center：居中对齐。
  • space-between：两端对齐，项目间隔相等。
  • space-around：两端对齐，项目间隔相等，项目间的间隔大于两端。
  • space-evenly：项目间间隔相等。

四、flex简写属性

flex 属性是 flex-grow、flex-shrink 和 flex-basis 的简写。

  • flex: 1 1 100px：表示弹性元素可以增长和收缩，初始大小为 100px。
  • flex: 0 0 auto：不允许元素伸展或收缩，初始大小由内容决定。
